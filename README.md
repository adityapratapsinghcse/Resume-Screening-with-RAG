# AI Resume Shortlister

An AI-powered resume shortlisting system that analyzes and ranks resumes based on relevance to job descriptions using Retrieval-Augmented Generation (RAG) techniques.

---

## Features

- Resume ranking and matching
- AI-powered relevance analysis
- Semantic embeddings
- Resume compression pipeline
- Retrieval-based candidate filtering
- Modular architecture

---

## Tech Stack

- Python
- NLP
- RAG Pipeline
- Embeddings
- Vector Retrieval

---

---

## How It Works

1. Resume data is processed
2. Embeddings are generated
3. Relevant candidates are retrieved
4. AI ranks resumes based on similarity and relevance
